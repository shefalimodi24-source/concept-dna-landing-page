# Concept DNA

**The Google Maps of STEM Learning**

Concept DNA is an AI-powered navigation system for STEM education that helps students identify what they should learn, when they should learn it, and whether they have the prerequisite knowledge needed to achieve their goals.

Instead of overwhelming learners with more content, Concept DNA provides personalized guidance through assessments, knowledge mapping, and learning roadmaps.

---

## Problem

College students today have access to thousands of courses, tutorials, research papers, and educational resources. However, finding content is no longer the challenge.

The real challenge is:

* What should I learn?
* What prerequisites do I need?
* Am I ready for this topic?
* What should I learn next?
* Which concepts are holding me back?

Concept DNA addresses these questions by transforming STEM learning into a structured and personalized journey.

---

## Features

### Goal-Based Learning

Students begin by selecting a target field such as:

* Natural Language Processing
* Machine Learning
* Robotics
* Computer Vision
* Data Science
* Cybersecurity

### Diagnostic Assessment

Students complete a readiness assessment tailored to their selected learning goal.

### STEM DNA Report

Concept DNA generates a personalized profile highlighting:

* Strengths
* Weaknesses
* Knowledge gaps
* Readiness scores
* Recommended next concepts

### Knowledge Graph

Visualize STEM concepts as an interconnected network and understand prerequisite relationships between topics.

### Personalized Roadmaps

Generate structured learning paths that guide students from foundational concepts to advanced topics.

### Progress Tracking

Track learning milestones and identify the highest-impact concepts to learn next.

## Vision

Our vision is to build a personalized navigation layer for STEM education.

As technology evolves and new fields emerge, students need more than access to content—they need guidance. Concept DNA aims to become the platform that helps learners confidently navigate complex STEM learning journeys and achieve their academic and professional goals.

## Running Locally

Clone the repository:

```bash
git clone https://github.com/shefalimodi24/concept-dna.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

in your browser to view the application.
